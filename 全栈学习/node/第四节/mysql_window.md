## 启动停止


```shell
# window 下启动 mysql
# cmd 管理员模式下，进入到 mysql bin目录下
$ net stop mysql80	# 停止 	mysql80是服务名称
$ net start mysql80	# 启动	mysql80是服务名称


# 直接 ctrl + shift + . 在服务中找到对应服务右键启动
```





## 进入 mysql 命令模式

```shell
# 进入到 mysql bin目录下
mysql -u root -p

# 输入密码
```
