# 项目结构

```js
+ pages
	+ index
		index.js		// 
		index.json	// 配置文件
		index.wxml
		index.wxss
	+ logs

+ utils
app.js		// 逻辑入口文件
app.json	// 全局配置文件
app.wxss	// 全局样式
project.config.json		// 配置小程序是否允许被索引
```



## app.json

```js
{
  "pages": [		// 配置页面路由
    "pages/index/index",	// 数组中第一项代表的小程序首页
    "pages/logs/logs"
  ],
  "window": {		// 设置小程序窗口的一些值
    "backgroundTextStyle": "light",
    "navigationBarBackgroundColor": "#fff",
    "navigationBarTitleText": "WeChat",
    "navigationBarTextStyle": "black"
  },
  "sitemapLocation": "sitemap.json"		//
}
```



**页面和组件中的 .json .wxss可以覆盖全局的**
