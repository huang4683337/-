## 国外服务bbr加速

```shell
# 首先登陆到国外服务 root 用户
$ ssh root@IP

# 粘贴一下代码
wget --no-check-certificate https://github.com/teddysun/across/raw/master/bbr.sh && chmod +x bbr.sh && ./bbr.sh

# 显示 Press any key to start...or Press Ctrl+C to cancle
# 输入任意键继续

# 显示 Do you want to restart system
# 输入 y 重启

# 稍等几分钟，等待服务重启完成
# 重启后再次连接国外服务root用户

# 输入指令查看版本
$ uname -r	# 2.6.32-754.25.1.el6.i686 返回类似结果


```
