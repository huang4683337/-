## 词法作用域

+ 编写代码时，在那个`{}` 写的就属于哪个词法作用域

+ 每个 `{}` 形成一个气泡

+ 作用域气泡的结构和相互之间的位置关系，给引擎提供了足够的位置信息。引擎通过位置关系逐级向上查找变量的引用，找到就停止。



## 欺骗词法

### eval

`eval(..)` 函数可以接受一个字符串为参数，并将其中的内容视为好像在书写时就存在于程序中这个位置的代码。

```js
function foo(str, a) {
 	eval( str ); // 欺骗！
 	console.log( a, b );
}
var b = 2;
foo( "var b = 3;", 1 ); // 1, 3
```

+ 相当于 `str` 对应的代码，就写在 `eval` 的位置

+ `eval` 将代码伪装成它所在的位置
