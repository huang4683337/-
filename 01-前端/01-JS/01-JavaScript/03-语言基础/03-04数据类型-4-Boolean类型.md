## Boolean 类型

| 数据类型  | 可以转换为 true 的值     | 可以转换为 false 的值 |
| --------- | ------------------------ | --------------------- |
| Undefined | N/A(不存在)              | undefined             |
| Boolean   | true                     | false                 |
| Number    | 非零的数值（包括无穷值） | 0｜NaN                |
| String    | 非空字符串               | ""（空字符串）        |
| Object    | 任意对象                 | null                  |
