# 关系型数据库和非关系型数据库

[菜鸟教程](https://www.runoob.com/mongodb/nosql.html)

## 关系

关系就是表，就是表与表之间的存在的关系

- 所有的关系型数据库都需要通过`sql`语句来操作
- 所有的关系型数据库在操作之前都需要设计表结构
- 数据表支持约束
  + 唯一的 
  + 主键
  + 默认值
  + 非空



## 非关系

+ 非关系型数据库非常的灵活
+ 有的非关系型数据库就时 key-value 对
+ MOngoDB是长得最像关系型数据库的非关系型数据库
  - 数据库 ==> 数据库
  - 数据表 ==> 集合（数组）
  - 表记录 ==> (文档对象)

+ MongoDB不需要设计表结构，可以任意在里面存数据，没有结构性这种说法
