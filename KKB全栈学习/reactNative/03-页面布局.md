## auto 属性

```js
//flex 布局，左边连个紧挨着， 右边单独一个

    ul{
        display: flex;

    }

    li{
        background-color: red;
        width: 100px;
        height: 100px;
        list-style: none;
    }


    <ul>
        <li></li>
        <li style='margin-right:auto'></li>
        <li></li>
    </ul>


// 使用 margin-right:auto 占用右边所有部分
```
