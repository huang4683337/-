## 

==相对路径==

```javascript
// 在 node 中, 一般是相对于启动当前终端的所在路径

/*
node/data/index.js

比如我在 data 目录下启动终端：路径就可以是 ./index

我们在 node 目录下启动终端：路径就是 ./data/index.js
*/

```



==绝对路径==

```javascript
绝对路径:
c:data/index.js  // 但是当项目或者文件变了位置 就不能用了

或者

/data/index.js		// 当前文件所在的磁盘根目录
```



==解决方案==

```javascript
将相对路径变为绝对路径
```
