# 水平垂直居中

## 定位（父元素、子元素宽高固定）

```css
	<div class='Father'>
        <div class='children'></div>
	</div>
<style>
   .Father{
     position: relative;
   }
   .children{
     width: 50px;
     height: 50px;
     position: absolute;
     top: 0px;
     bottom: 0px;
     left: 0px;
     right: 0px;
     margin: auto;
   }
</style>
```



## flex (父子元素宽高不定)

```css
    <div>
        <div></div>
    </div>
  <style scoped>
  .Father {
    border: 1px solid red;
    height: 100px;
    display: flex;
    justify-content: center;
    align-items: center;
  }
  .children {
    width: 50px;
    height: 50px;
    border: 1px solid blue;
  }
</style>
```



## transform (父子元素宽高不定)

```css
/* 子元素绝对定位  transform进行调整*/
<div>
        <div></div>
</div>
  
<style scoped>
  .Father {
    border: 1px solid red;
    height: 100px;
    position: relative;
  }
  .children {
    width: 50px;
    height: 50px;
    border: 1px solid blue;
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);  /* 使用css3的transform来实现 */
  }
</style
```
